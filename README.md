# 如何使用？

html代码：

```
<!doctype html>
<html lang="zh-CN">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>中文编辑器示例</title>
    <link rel="stylesheet" href="editor.css" />
  </head>
  <body>
    <div id="editor"></div>
    <script src="editor.js"></script>
  </body>
</html>
```

只需要在需要的地方使用<div id="editor"></div>，之后引入css和js。

# 说明：
板块和阅读限制可在editor.js中搜索关键词替换为你自己的提交参数，也可自行删除。
编辑器提交的参数可以通过F12控制台查看json格式，自行根据json格式提交对应数据。
